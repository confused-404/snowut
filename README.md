# SnowUT

A Vite + React website for the TSA webmaster challenge. The site helps snow sport community members find safety resources, adaptive programs, youth access, climate organizations, events, financial support, volunteer opportunities, and community services.

## Features

- Searchable and filterable resource directory
- Category, region, and audience filters
- Expandable resource cards with contact and impact details
- Auto-rotating spotlight carousel
- Events calendar preview
- Region-based interactive map
- Volunteer matching tool
- Resource guide content
- Submit-a-resource form with validation and confirmation

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
